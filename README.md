## spring 기록1

강의제목 : spring 핵심 원리 - 기본편 (김영한)

스프링 강의를 보고 기록한 소스코드 입니다.  

2023 03 05 시작
2023 04 30 종료

### 요구사항
- 회원을 가입하고 조회 사능
- 회원은 일반과 vip 등급
- 회원 데이터는 자체 db를 구축할 수 있고, 외부 시스템과 연동 가능

### 학습 포인트
- 객체지향의 다형성
- DI 컨테이너, 의존성, 의존성 주입
- 스프링 = 싱글톤 컨테이너
- 컴포넌트 스캔
- 의존관계 자동주입
- 빈 생명주기 콜백 및 빈 스코프

### 반성점
- 강의를 듣는 시간을 줄일 필요가 있음.
- 12시간 정도의 강의임에도 2달의 시간이 걸림.
- 학기중임을 감안해도 너무 오래걸렸다고 생각함.
- 24시간 강의 기준 1달에 1개 정도의 강의마무리가 필요하다고 생각.
